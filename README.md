# Quiz Web Marketing

App statica per ripassare l'esame di Web Marketing e Comunicazione Digitale.

- 500 domande a risposta multipla
- 50 domande casuali per sessione
- modalita per argomento
- ripasso errori e salvataggio locale dei progressi

Il banco domande e generato da `build-questions.mjs` e validato con `validate-questions.mjs`.
